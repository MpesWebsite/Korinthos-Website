Ανοιχτά Δεδομένα

Κατηγορίες Δεδομένων

  • Δημογραφικά (πληθυσμός, ηλικιακή κατανομή)
  • Οικονομικά (προϋπολογισμός, έσοδα–έξοδα)
  • Υποδομές (έργα, φάσεις υλοποίησης)

API & Λήψη Αρχείων

Τα σύνολα δεδομένων είναι διαθέσιμα σε CSV, JSON και XML. Το API επιτρέπει πρόσβαση query-based:

GET https://data.korinthos.gov/api/v1/demographics?year=2023
